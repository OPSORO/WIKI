config(pin=None, min_range=0, mid_pos=1500, max_range=0)[source]¶

Helper class to turn DOF positions into pulse widths for the servo controller.

Arguments:
pin (int): Servo pin number min_range (int): Minimum range of the servo, can be positive or negative. When dof_pos < 0, pulse width = mid_pos + dof_pos*min_range mid_pos (int): Pulse width when neutral (DOF position = 0). max_range (int): Maximum range of the servo, can be positive or negative. When dof_pos > 0, pulse width = mid_pos + dof_pos*max_range
